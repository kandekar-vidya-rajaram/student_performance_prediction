# 🎓 Student Academic Performance Prediction

## 📌 Project Overview

This project is a **Machine Learning-based system** designed to predict student academic performance. It classifies students into categories such as **At Risk, Average, or Excellent** based on academic data.

The goal is to help educators identify weak students early and provide necessary support.

---

## 🚀 Features

* 📊 Student performance prediction (At Risk / Average / Excellent)
* 🤖 Machine Learning model (Random Forest Classifier)
* 📉 Model evaluation using Accuracy & Confusion Matrix
* 📂 Data preprocessing and cleaning
* 🧠 Automated analysis based on input features

* 📉 Confusion Matrix

## 🧠 Machine Learning Model

* Algorithm Used: **Random Forest Classifier**
* Why Random Forest?

  * Handles large datasets efficiently
  * Reduces overfitting
  * High accuracy for classification problems

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/student_performance_prediction.git
```

### 2️⃣ Navigate to project folder

```
cd Student-academic-performance-prediction
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run backend server

```
python app.py
```

### 5️⃣ Run frontend

```
npm install
npm start
```
